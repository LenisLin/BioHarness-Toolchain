# BioHarness-Toolchain

BioHarness-Toolchain is a tools and infrastructure repository within the broader BioHarness framework.

This repository focuses on tools and infrastructure only. Audit and governance concerns are out of scope here. Upper-level reasoning and brain responsibilities are also out of scope for this repository.

The first vertical is spatial transcriptomics downstream analysis.

## Status

Initialization stage. Repository skeleton, placeholder documentation, and minimal CI are present. Detailed tool design and implementation remain open.
